# Weather App 🌤

Aplicación Flutter para consultar el clima en tiempo real mediante integración con API externa.  
Permite buscar por ciudad y mostrar datos meteorológicos actualizados con una interfaz intuitiva.

## 🚀 Tecnologías
- Flutter
- API OpenWeather
- Dart
- JSON
